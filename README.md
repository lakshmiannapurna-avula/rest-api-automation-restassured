# REST API Automation Framework

API automation framework built using Java, Spring, REST Assured, TestNG, and Maven.

## Overview
This project demonstrates backend API automation with reusable framework design, database validation utilities, and assertion-based validation. It is designed to reflect real-world enterprise automation practices used in large-scale applications.

## Features
- API testing using REST Assured
- Test execution using TestNG
- Reusable base API setup
- Validator-based response and database checks
- Database utility methods using JDBC
- Logging support with Log4j
- Maven-based build and execution

## Tech Stack
- Java 17
- Spring Core
- Spring JDBC
- REST Assured
- TestNG
- Maven
- MySQL
- Log4j
